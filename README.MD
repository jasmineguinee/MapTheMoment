# MapTheMoment

[Render deployed website](https://mapthemoment.onrender.com)  

MapTheMoment is a Placemark Hapi/Node Application which allows users to log wedding venues and proposal hotspots. 

## Description
Users can create areas and add venues for weddings or proposals. 
Users can decide to make venues publicly visible or they can keep them private. 
On the main dashboard when the user logs in, they can view all public postings for wedding venues or proposal hotspots, choosing which catagory they wish to see. When the user clicks on their area, they have their private map which shows the venues they have added for that area.

## Use cases for application
This app can be used for businesses to display information about and promote their venues. Users can also post venues and proposal hotspots they have found and give their experience in the description. This could also be helpful for users who are seeking to plan events and wish to access to all information in one spot.
